A|T = TA / T

A=.1
P=.01

T|~P~A = .1
T|~PA = .5
T|P~A = .6
T|PA = .9

~P~A = .99*.9 = .891
~PA = .99*.1 = .099
P~A = .01*.9 = .009
PA = .01*.1 = .001

ATP = .9*.001 = .0009

1)A|T
T = T|PA * PA + T|~PA * ~PA + T|P~A * P~A + T|~P~A * ~P~A
= .1*.891 + .5*.099 + .6*.009 + .9*.001 = .1449

TA = TA~P + TAP = T|A~P * A~P + ATP
= .5*.099 + .0009 = .0504

A|T = AT / T = .0504 / .1449 = 0.34782608695652173

2)A|TP
A|TP = ATP / TP = ATP / (ATP + TP~A)
= .0009 / (.0009 + .6*.009) = 0.14285714285714288
